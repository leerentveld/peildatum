# peildatum
Files for the Bitcoin [peildatum](https://peildatum.nl) site.

### average.py
Set the `input_date` and `input_file` in the script and execute it.

Example output:
```
./average.py
----------------------------------------
Date    : 2019-01-01
File    : bitstampEUR.csv
Average : 3255.12
----------------------------------------
Date    : 2019-01-01
File    : coinbaseEUR.csv
Average : 3256.52
----------------------------------------
Date    : 2019-01-01
File    : krakenEUR.csv
Average : 3257.04
----------------------------------------
Total average value: 3256.23
----------------------------------------
```
